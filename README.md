Descrição do Projeto: 
O Sistema de Controle de Atrasos Escolar foi desenvolvido para modernizar o registro e o acompanhamento de atrasos dos alunos. O projeto integra uma API em Java hospedada no Render e um aplicativo criado no FlutterFlow, permitindo que usuários realizem login, cadastros e consultem informações de forma simples e acessível, com navegação fluida entre as telas. 

Funcionalidades: 
O sistema oferece um CRUD completo, incluindo cadastro, login, listagem, busca por ID, atualização e exclusão de dados, tudo por meio de uma API REST. No front-end, o FlutterFlow fornece telas de login, cadastro, listagem e detalhes do usuário, todas integradas à API por meio de chamadas diretas, garantindo que as informações sejam atualizadas em tempo real.
